# Claude Memory Lane

This directory is Claude's lane in the shared cross-tool memory pattern between Cofounder and Claude for Anchor Marianas.

## Write here
- Technical decisions made in Claude Code sessions
- Implementation notes, architecture choices, what was tried and rejected
- Open technical questions for Cofounder agents to consider
- Session summaries from substantive Claude conversations (Code or Chat)

## Read both lanes
- Before starting substantive work, read `../cofounder/` for business context
- After major work, append a summary to `claude/session-summaries.md` (create it on first write)

## File conventions
- All markdown
- Append-only logs preferred (newest entries on top, dated)
- Commit context updates alongside code changes when both happen in the same session
